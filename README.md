\# 🚀 Terraform Specialist: Cloud Provisioning on GCP



\*\*🇺🇸 English:\*\* Documentation of my journey in mastering Infrastructure as Code (IaC) specifically for Google Cloud. This repository focuses on automated, secure, and resilient infrastructure.



\*\*🇧🇷 Português:\*\* Documentação da minha jornada na especialização de Infraestrutura como Código (IaC) para Google Cloud. Este repositório foca em infraestrutura automatizada, segura e resiliente.

\## 🛠️ Key Learning Modules / Módulos de Aprendizado



\- \*\*IaC Fundamentals:\*\* Workspaces, Remote State (GCS), and Provider configuration.

\- \*\*Secure Networking:\*\* Provisioning VPCs, Shared VPCs, and Cloud Armor via TF.

\- \*\*Automated Resilience:\*\* Deploying Managed Instance Groups (MIGs) with Auto-healing.

\- \*\*Identity as Code:\*\* Managing IAM Bindings and Service Accounts programmatically.
